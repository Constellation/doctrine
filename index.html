doctrine (doctrine) is a JSDoc parser.

It's used by

  • Eclipse Orion's content assist system (detail)
  • eslint's valid-jsdoc rule

Doctrine can be used in a web browser:

<script src="doctrine.js"></script>

or in a Node.js application via the package manager:

npm install doctrine

simple example:

doctrine.parse(
    [
        "/**",
        " * This function comment is parsed by doctrine",
        " * @param {{ok:String}} userName",
        "*/"
    ].join('\n'), { unwrap: true });

and gets following information

{
    "description": "This function comment is parsed by doctrine",
    "tags": [
        {
            "title": "param",
            "description": null,
            "type": {
                "type": "RecordType",
                "fields": [
                    {
                        "type": "FieldType",
                        "key": "ok",
                        "value": {
                            "type": "NameExpression",
                            "name": "String"
                        }
                    }
                ]
            },
            "name": "userName"
        }
    ]
}
